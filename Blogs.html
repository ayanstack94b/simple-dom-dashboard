Question-1: What are the different ways to select an element in the DOM?

You can select elements in DOM in few ways. getElementById() find by ID, getElementsByClassName() find by class, and getElementsByTagName() find by tag. querySelector() and querySelectorAll() are more flexible for selecting elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML show all HTML inside element, so it can change tags too. innerText only show visible text, it ignore hidden text and spaces. textContent show all text, even hidden ones, but no HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is when u put event on parent instead of many child. If child clicked, event go up to parent and run. This help when many child or new child added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is when event start from clicked element and go up to parent, then grandparent and so on. It happen auto in DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you make element with document.createElement(), put it in page with appendChild() or append(), and remove it with remove().
